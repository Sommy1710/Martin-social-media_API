Social Media API
🎯 Overview
This project is an API for a social media platform. It allows users to create, read, update, and delete posts, also like, comment, delete, users can also reply to comments, and like reply to comments, also delete comments.
Users can also send and recieve messages.
The main objective is to provide a simple, yet powerful backend for a social media platform that developers can integrate with any frontend.

✨ Features
Register and login: Users can register and log in
Create Posts: users can create posts.
Update Posts: Modify existing posts with new information.
Delete Posts: Remove posts from the platform.
View Blog Posts: Retrieve single or multiple posts.
Like posts: Users can like posts.
Comment on Posts: Users can comment on posts.
Reply to comments: Users can reply to comments.
Like reply to comments: Users can like comment reply.
Delete Comment: Authenticated users can delete comment.
Send and Recieve messages: Users can send and recieve messages.
JSON API: Easily integrates with any frontend.

🚀 Getting Started
To get a local copy up and running, follow these steps:

Prerequisites
Ensure you have the following installed:

Node.js
npm (Node Package Manager)
Express
MongoDB
Git
Installation
Clone the repository:

git clone 
Install dependencies:

npm install
Create a .env file in the root directory with the following content:

see .env.example
Start the development server:

npm run start:dev
